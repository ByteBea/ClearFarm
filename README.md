# ClearFarm  

### Obiettivo
Questo progetto utilizza la tecnologia blockchain per tracciare la filiera agricola, garantendo trasparenza, sicurezza e autenticità dei prodotti e facilitandone la vendita.  

### Funzionalità principali
- Tracciamento delle colture
- Registrazione delle transazioni agricole
- Certificazione dei prodotti

### Tecnologie Utilizzate
- Python
- Solidity
- Avalanche
